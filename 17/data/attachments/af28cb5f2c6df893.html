  • JUnit 5 — это последняя версия фреймворка JUnit для модульного тестирования Java-приложений.

    Она состоит из трёх модулей:

    1. JUnit Platform — основной модуль для управления тестами.
    2. JUnit Jupiter — модуль, который использует новые возможности Java 8. Он предоставляет API на основе аннотаций и позволяет работать с модульными и динамическими тестами.
    3. JUnit Vintage — модуль для поддержки тестов, написанных с использованием JUnit 3 и JUnit 4.

    JUnit 5 позволяет гибко указывать условия тестирования, объединять тесты в группы, распределять их по функциональности, тестируемым модулям или уровню критичности, прописывать условия запуска для каждого блока кода и анализировать результаты по отдельности.
